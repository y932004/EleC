# Campus Broomstick Chase Web App

## Test locally
Use a local HTTP server rather than opening index.html as a file:

```bash
python3 -m http.server 8080
```

Then open the local address shown by the server.

## Deploy
Upload the entire folder to any HTTPS host such as GitHub Pages, Netlify, or Vercel.

## Install on iPhone
Open the deployed HTTPS page in Safari, tap Share, then Add to Home Screen.

## Install on Android
Open the deployed HTTPS page in Chrome, then choose Install app or Add to Home screen.
